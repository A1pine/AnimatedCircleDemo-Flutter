# AnimatedCircleDemo-Flutter
 A flutter implementation of Kitten Yang's GooeyCircle
## 目的
 这个项目的~~本质就是复读机~~目的就是将 Kitten Yang 大神为 iOS 写的动画效果教程
A Guide To iOS Animation 的 demo 移植到跨平台的 Flutter 上
由于 flutter 的控件也用了 canvas 作为自绘控件的接口,所以项目初期会尝试使用 html + css + js的组合在浏览器上实现对应特效, 第二步再尝试将动效移植到 flutter 上去.
## 起因
我真的太爱 Kitten Yang 大神写的 Cosmos 微博客户端了,可惜现在这款客户端已经下架了
正好我想学动画效果和 flutter, 再加上 html 也有后就没复习了,干脆就带着一起写了
初步翻了一下书上教到的特效大概有 20 个.每一个我都会尝试将它写透并且尽量保证不会出现 bug. 也就只有假期能有完整的时间,我准备抽空慢慢写了,项目的周期大概在一年左右吧
